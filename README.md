# Pinneed
**Pinneed** is a MERN-based chat application with a unique twist. Unlike modern messaging platforms that deliver messages instantly, **Pinneed** intentionally delays every message by **24 hours**.

The idea behind Pinneed is to bring back the excitement, anticipation, and emotional value of waiting for a message—something that has gradually disappeared in today's era of instant communication.

By combining nostalgic human experiences with modern technology, Pinneed encourages users to slow down, be more intentional with their words, and rediscover the joy of looking forward to a conversation.
